## 📖 О проекте

**Hozz** — это современное десктопное приложение, созданное для полного контроля над сетевым трафиком и безопасного управления инфраструктурными секретами. Проект спроектирован с упором на максимальную производительность, безопасность и удобство использования.

Дизайн интерфейса вдохновлен философией **Vercel** и **Linear**: глубокий черный, цинковый серый, полупрозрачные стеклянные элементы (*Glassmorphism*) и строгий минимализм, благодаря чему приложение ощущается легковесным и быстрым.

## 🚀 Ключевые особенности

- **Управление прокси на базе ядра Mihomo:** Эффективная маршрутизация соединений с возможностью точечного управления прокси-узлами (отключение нерабочих серверов через встроенный Blacklist-менеджер в подписках).
- **Интеграция с HashiCorp Vault:** Превращает приложение в полноценный центр управления секретами (Secret Ops). Поддерживает просмотр дерева секретов, парсинг временных директорий, отслеживание времени жизни сессии (TTL) и генерацию wrap-ссылок.
- **Интерактивный сетевой журнал (Аналитика трафика):** Продвинутая фильтрация сетевых адресов в реальном времени, автоматическое детектирование и блокировка систем телеметрии и трекеров, фоновое обнаружение «потеряшек».
- **Инженерная надежность:** - Автоматический мониторинг и health-check TUN-интерфейса с мгновенным восстановлением.
  - Фоновый сбор телеметрии с контролем утечек памяти (Zero Memory Leak).
  - Механизм Single Instance Application (повторный запуск разворачивает существующий инстанс).
  - Кастомная системная панель (Tray Icon) с поддержкой Linux (AppIndicator/KStatusNotifier), macOS и Windows.
- **Автозагрузка:** Нативная интеграция автозапуска при старте ОС (интеграция с реестром Windows и `.desktop`-файлами Linux).

## 🛠 Технологический стек

- **Язык:** Rust 🦀 (высокая производительность, безопасность памяти, отсутствие сборщика мусора).
- **Графический интерфейс (GUI):** Dioxus + Inline SVG ( Material Design Icons ), запеченные шрифты/иконки для оффлайн-работы без внешних зависимостей.
- **Сетевое ядро:** Mihomo (управление TUN-режимом и проксированием).
- **Хранилище конфигураций:** Sled (высокопроизводительная локальная БД) + Figment (гибкая сборка конфигураций).
- **Безопасность:** Использование `secrecy::SecretString` для защиты токенов и секретов.

## 🏗 Архитектура

Ядром приложения является масштабируемый **Оркестратор (Orchestrator)**. Архитектура построена на базе **Трейтов (Traits)** и разделения ответственности:
- Модуль `machine` инкапсулирует сбор системных ресурсов.
- Сетевая логика, управление хранилищем Sled и сессиями HashiCorp Vault строго изолированы. Это позволяет легко расширять функционал, не превращая приложение в «божественный объект» (*God Object*).

## 🪟 Системные требования и сборка

Приложение является кроссплатформенным и собирается под:
- **Windows:** Нативная поддержка, включая корректное позиционирование окон при смене DPI, правильное расширение бинарников (`.exe`) и обход блокировок TUN-адаптера за счет Hyper-V.
- **Linux:** Полная адаптация системных треев, инлайн-декорации окон и интеграция Dioxus через DBus/AppIndicator.
- **Android:** Экспериментальная поддержка сборки под мобильную архитектуру Activity/WebView.
